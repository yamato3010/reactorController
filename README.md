# ReactorController

## 概要

私がVRChatで使用しているアバターの胸部に装着されているリアクターを制御するためのソフトウェアです。OSCを用いて制御を行っています。OSCによるアバターの制御を学習する目的で作成しました。

## 機能

- 手動操作によるリアクターの「点灯」「消灯」「点滅」状態の切り替え
- 指定した時刻にリアクターの状態を変更
